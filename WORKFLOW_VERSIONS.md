# Workflow versions

The workflow experiments are organized into three top-level folders:

- `workflow-v1/`
- `workflow-v2/`
- `workflow-v3/`

Each workflow folder has the same structure:

```text
workflow-v*/
├── data/
├── code/
├── results/
└── README.md
```

For the complete baseline + workflow code/result map, read
`EXPERIMENT_INDEX.md`.
